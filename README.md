# 🧺 Laundry App – Login Page UI

A modern and clean **Laundry App Login Page UI** designed using **HTML & CSS**.  
This project focuses on a minimal, user-friendly authentication screen with a fresh blue theme and 3D illustration.

---

## ✨ Features

- Clean & modern UI design  
- Responsive login layout  
- Email & password input fields  
- “Forgot Password” option  
- Call-to-action **Login** button  
- Simple and readable design structure
